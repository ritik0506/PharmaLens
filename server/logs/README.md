# PharmaLens Audit Log
# This directory contains compliance audit logs

This directory stores audit logs generated by the Winston logger.

## Log Files

- `combined.log` - All log entries (info, warn, error)
- `error.log` - Error-level logs only
- `exceptions.log` - Uncaught exceptions
- `rejections.log` - Unhandled promise rejections

## Log Format

Logs are stored in JSON format for easy parsing:

```json
{
  "level": "info",
  "message": "Research analysis initiated",
  "service": "pharmalens-server",
  "timestamp": "2024-01-15T10:30:00.000Z",
  "event": "RESEARCH_STARTED",
  "molecule": "Metformin",
  "mode": "secure",
  "requestId": "uuid-here"
}
```

## Retention

Logs are automatically rotated when they reach 10MB.
Maximum of 5 rotated files are kept.

## Compliance

These logs support:
- HIPAA audit requirements
- FDA 21 CFR Part 11 compliance
- SOX controls
- GDPR data processing records
